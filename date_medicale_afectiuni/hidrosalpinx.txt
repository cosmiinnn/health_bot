Hidrosalpinxul este o patologie a trompelor uterine si consta in blocarea acestora cu lichid. Consecinta acestei patologii este infertilitatea sau sarcina ectopica. Frecvent, este asimptomatica si, ca urmare, diagnosticul este obtinut doar cand se constata nereusita obtinerii unei sarcini. Tratarea hidrosalpinxului creste sansele de a ramane insarcinata.
Ce este hidrosalpinxul?
Prin hidrosalpinx se intelege blocarea cu lichid a trompelor uterine. Adesea, sunt blocate ambele trompe, dar lichidul poate afecta si o singura trompa.
Hidrosalpinxul este raspunzator pentru aproximativ 20% dintre cazurile de infertilitate. Blocarea trompelor poate avea loc la nivelul:
capatului dinspre ovar (cea mai frecventa situatie);
capatului dinspre uter;
ambelor capete.
Hidrosalpinxul afecteaza preponderent fertilitatea femeii. Trompele uterine reprezinta locul de pasaj pentru ovocit si spermatozoizi. Ovocitul este eliberat de catre ovar in cavitatea abdominala, de unde este preluat de trompele uterine si, astfel, ajunge la nivelul uterului. Spermatozoizii patrund in vagin, trec prin uter si ajung la nivelul trompelor uterine. De cele mai multe ori, fertilizarea ovocitului se realizeaza in lumenul trompelor uterine.
Daca lumenul este blocat, se pot intalni doua situatii:
ovocitul si respectiv spermatozoizii nu se pot intalni; ca urmare, embrionul nu se formeaza, situatie cunoscuta drept infertilitate;
ovocitul si spermatozoidul formeaza embrionul, dar, din cauza obstructiei, acesta nu poate ajunge in cavitatea uterina si se implanteaza la nivelul trompelor uterine, generand o sarcina extrauterina.
Hidrosalpinx: cauze
In cele mai multe dintre situatii, cauza principala a hidrosalpinxului este reprezentata de o infectie ce nu a fost tratata corespunzator. Principalele infectii responsabile de aparitia hidrosalpinxului sunt cele cu transmitere sexuala. Frecvent, sunt implicati urmatorii agenti patogeni:
stafilococul;
streptococul;
gonococul;
Escherichia coli;
Chlamydia sp.
Alte cauze ale hidrosalpinxului sunt urmatoarele:
interventiile chirurgicale;
formarea aderentelor;
inflamatiile;
endometrioza
;
cancerul ovarian;
cancerul trompelor uterine.
Hidrosalpinx: simptome
De obicei, hidrosalpinxul este o afectiune asimptomatica. In consecinta, diagnosticul se pune in situatia in care sunt efectuate investigatii pentru a determina cauzele
infertilitatii
sau daca se formeaza o sarcina ectopica.
Studiile au demonstrat ca femeile care sufera de hidrosalpinx pot experimenta semne si simptome nespecifice, precum:
scurgeri vaginale;
disconfort sau durere abdominala pe durata
menstruatiei
;
febra.
Hidrosalpinx: diagnostic
Diagnosticul hidrosalpinxului se face prin realizarea investigatiilor imagistice.
Astfel, medicul poate recomanda urmatoarele:
histerosalpingograma - aceasta investigatie consta in administrarea unei substante de contrast si in evidentierea traseului acesteia, prin intermediul unei radiografii. Pentru obtinerea rezultatelor se folosesc radiatiile ionizante. Medicul poate vizualiza atat uterul, cat si trompele uterine.
histerosalpingografia - este un tip de ecografie. Prin intermediul acesteia, este observat in mod direct uterul, dar se pot determina si eventualele obstacole de la nivelul trompelor uterine.
ecografia - este o metoda neiradianta, care permite vizualizarea trompelor uterine, a uterului si a ovarelor. Se poate determina si gradul de severitate al hidrosalpinxului.
tomografia computerizata (CT) - este o metoda iradianta, dar care permite vizualizarea trompelor uterine.
rezonanta magnetica nucleara (RMN) - este rar utilizata, dar, prin rezultatele oferite, se obtin informatii relevante pentru stabilirea diagnosticului.
In anumite cazuri, investigatiile imagistice nu sunt suficiente. Astfel, este recomandata laparoscopia. Aceasta este o metoda chirurgicala ce permite explorarea organelor reproducatoare. Pentru aceasta, sunt realizate cateva incizii la nivelul abdomenului, prin intermediul carora este introdusa o camera ce ajuta la vizualizarea organelor. Avantajul acestei investigatii este ca poate fi folosita si in scop terapeutic.
Hidrosalpinx: tratament
Tratamentul este recomandat in functie de starea de sanatate a fiecarei paciente. Dintre posibilele variante de tratament fac parte:
tratamentul laparoscopic - prin intermediul laparoscopiei pot fi excizate adeziunile formate in cadrul altor interventii chirurgicale.
salpingectomia - aceasta interventie chirurgicala presupune eliminarea partiala sau totala a trompei uterine afectate.
salpingostomia - in cadrul acestei interventii chirurgicale, se realizeaza o stoma (incizie) la nivelul trompei uterine. In acest mod, lichidul din lumenul trompei poate fi drenat.
aspiratia transvaginala - este o metoda minim invaziva. Efectuarea acestei interventii se face  sub ghidaj ecografic.
tratamentul medicamentos (pentru infectii se recomanda
antibioticele
specifice agentului patogen).
Hidrosalpinx: posibile complicatii
Principala complicatie a hidrosalpinxului este reprezentata de infertilitate. Daca hidrosalpinxul nu este tratat, sansele de a obtine o sarcina sunt reduse. Daca sarcina se obtine, este necesar a fi monitorizata inca de la inceput, pentru a diagnostica din timp posibilele complicatii, precum:
sarcina ectopica
;
avortul spontan.
Alte complicatii ce pot aparea, daca hidrosalpinxul nu se trateaza, constau in:
disconfort sau durere la nivelul abdomenului inferior (in timpul menstruatiei sau independent de aceasta);
scurgeri vaginale cu miros neplacut.
Hidrosalpinxul este o afectiune a trompelor uterine, in care acestea sunt blocate cu lichid. Cauza principala este data de infectii, dobandite in mod frecvent prin contact sexual. Hidrosalpinxul conduce la infertilitate sau la aparitia unei sarcini ectopice. De aceea, pentru reusita obtinerii unei sarcini, este necesar ca aceasta patologie sa fie tratata conform indicatiilor medicului.
