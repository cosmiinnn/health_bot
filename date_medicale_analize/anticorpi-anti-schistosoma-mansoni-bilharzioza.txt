Descriere
Schistosoma mansoni este un parazit trematod (vierme), care poate infecta oamenii si determina o infectie, cunoscuta ca schistosomiaza. Frecvent, infectia este dobandita in urma contactului cu ape infestate. Larvele acestui parazit sunt eliberate de melcii de apa dulci. Acestea patrund prin piele (transcutanat) in organismul uman si cauzeaza infectia.
Schistosomiaza este una dintre cele mai intalnite boli tropicale, aflandu-se pe locul 3 dupa malaria si helmitiaza. Lasata netratata, infectia poate determina chiar si decesul. In tari, precum: Africa, Asia, Orientul Mijlociu, schistosomiaza prezinta un grad crescut de mortalitate si morbiditate.
Indicatii dozare anticorpi anti-Schistosoma mansoni
Analiza pentru determinarea anticorpilor anti-Schistosoma mansoni este indicata de catre medic persoanelor care prezinta simptome specifice si care prezinta factori de risc, precum:
calatorii recente in zone endemice;
lipsa imunizarii (vaccinului);
contactul cu diferite ape ce prezinta risc de infectare;
istoric de boala.
In functie de zona din organism unde larvele parazitului sunt localizate, simptomele specifice pot fi diferite. Astfel, daca larvele se afla la nivelul sistemului digestiv, in jurul intestinului, manifestarile pot fi reprezentate de:
durere abdominala;
stare de greata;
varsaturi;
modificari ale tranzitului intestinal (diaree);
febra;
frison;
stare generala alterata.
In schimb, cantonarea parazitului la nivelul sistemului urogenital (vezica urinara) este evidentiata prin simptome, precum:
disurie (disconfort la urinare);
hematurie (urina cu sange, urina are culoare roz-rosie, in cazul unei hematurii macroscopice);
scaun sanguinolent.
Analiza pentru dozarea anticorpilor anti-Schistosoma mansoni este indicata pentru:
diagnosticarea infectiei;
identificarea unei infectii anterioare.
Pregatirea pentru analiza
Obtinerea unui rezultat corect al acestei analize nu este influentat de alimentatie. Asadar, inainte de a merge la centrul de recoltare nu este necesara realizarea unei pregatiri speciale. Daca doresti sa beneficiezi de cat mai mult confort, in timpul procedurii de prelevare a probei este recomandat a purta o bluza cu maneca scurta. Astfel, punctia venoasa se realizeaza cu o mare usurinta, deoarece zona plicii cotului poate fi usor abordata.
Mod de recoltare
Prelevarea probei de sange consta in realizarea unei punctii venoase, o procedura cu o durata de cateva minute, care nu creeaza un disconfort semnificativ. Riscurile asociate asociate punctiei venoase sunt reduse, iar prin respectarea etapelor specifice si a indicatiilor personalului medical, complicatiile sunt diminuate considerabil. Pentru a preveni infectiile, zona plicii cotului unde se realizeaza punctia venoasa este dezinfectata cu alcool sanitar, iar la sfarsitul procedurii, este aplicat un pansament, pentru a preveni patrunderea agentilor microbieni. De asemenea, dupa ce proba este obtinuta, este indicat a apasa usor zona de realizare a punctiei, cu scopul de a opri sangerare si de a nu se forma echimoze (vanatai).
Recoltarea presupune efectuarea unor etape precise. La finalul prelevarii, se obtine o proba de sange venos, care se stocheaza intr-un recipient asemanator unui tub. Acesta este etichetat corespunzator si trimis la laboratorul de analize medicale, pentru a fi prelucrat.
Interpretare
Rezultatele sunt eliberate de catre laboratorul de analize medicale in cateva zile lucratoare. Poti beneficia de rezultate atat in format digital, cat si fizic. Indiferent de optiunea aleasa, este important a face o programare la medic pentru interpretarea valorilor obtinute.
Rezultat pozitiv
Un rezultat pozitiv confirma prezenta anticorpilor anti-Schistosoma mansoni in proba de sange. Aceasta poate indica:
infectie acuta;
infectie cronica;
infectie anterioara.
Diferentierea intre tipurile de infectie cu Schistosoma se face prin alte investigatii. De asemenea, cand este stabilit tratamentul corespunzator se ia in considerare si starea clinica a pacientului.
Rezultat negativ
In cele mai multe dintre situatii, un rezultat negativ infirma prezenta infectiei. Cu toate acestea, exista si persoane la care sistemul imunitar nu produce anticorpii specifici. In aceste cazuri, rezultatul obtinut este unul fals negativ. Daca simptomele sunt prezente, medicul poate indica efectuarea unor investigatii suplimentarea, pentru stabilirea unui diagnostic corect.
Analiza pentru determinarea nivelului de anticorpi anti-Schistosoma mansoni este o investigatie utilizata pentru diagnosticarea infectiei cu acest parazit. Pentru efectuarea acesteia este necesara o proba de sange, care se obtine printr-o procedura minim invaziva. Este important a retine faptul ca poti preveni schistosomiaza sau bilharzioza prin evitarea contactului cu ape contaminate si prin imunizarea adecvata. Comunica deschis cu medicul si afla care sunt recomandarile acestuia pentru a-ti mentine o stare optima de sanatate.
